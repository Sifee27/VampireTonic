## ğŸš€ Quick Start

### Prerequisites
- Ruby 3.4.1 or higher
- Jekyll 4.4.0 or higher
- Bundler

### Running Any Theme

1. Navigate to the theme directory:
   ```bash
   cd themes/[theme-name]
   ```

2. Install dependencies:
   ```bash
   bundle install
   ```

3. Serve the site:
   ```bash
   bundle exec jekyll serve --port [port-number] --host 0.0.0.0
   ```
